# Sandbox

A personal HTML/CSS/JS sandbox.

Try it out 👉 https://audinue.github.io/sandbox/

PR welcome!

### Features

- ✔ Transforms JSX and TypeScript
- ✔ Renders markdown
- ✔ File drop
- ✔ Preview images
- ✔ Persistent resizable layout
- ✔ Duplicate project or file
- ✔ Everything is done locally
